# Published website

This repository contains the generated static files for the Hereford Securities website.

Do not edit files directly.

The contents are generated by the `StaticSitePublisher` project in the
`HerefordSecurities` repository.

## Publishing

1. Run `StaticSitePublisher`.
2. Review the generated files.
3. Commit and push this repository.
4. GitHub Pages deploys automatically.
